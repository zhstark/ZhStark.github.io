我又回来了，整理一下我的 Mac 软件与服务配置，有时间开始写文章，之前打算记录一下自己的工作历程，还有之前在美国 Perry 教我的一些东西。

装机必备

  • Chrome，不必多说
  • iTerm2，终端软件
  • Ulysses，写作软件
  • Omnifocus，GTD，用于备忘、提醒、规划
  • PDF Expert，最好的 PDF 阅读编辑器
  • Day One，日（年）记软件
  • Manico，效率神奇
  • Alfred，效率神奇，搞了一些脚本，神器中的神器
  • iStat Menus，监控软件，强迫症必备
  • 欧陆词典，最好用的词典
  • IINA，视频播放器
  • Magnet，窗口管理工具
  • Tuxera Disk Manager，移动硬盘写入
  • Mielink，看看外面的世界
  • The Unarchiver，解压工具
  • 1Password，密码管理工具

效率工具

  • Fantastical，日历工具，买的有点亏
  • Popclip，方便复制文字打开链接
  • Bartender 4，强迫症必备
  • Amphetamine，防自动睡眠，睡觉下载专用
  • Xmind，思维导图
  • Mindnode，轻量级思维导图
  • Marked2，markdown 阅读器，其实没啥用，当初花钱买了不舍得弃用
  • Focus Timer，番茄时钟，不能一坐坐一天

开发工具

  • Docker compose
  • Intellig 全家桶
    • IDEA
    • PyCharm
    • DataGrip
  • CodeRunner，测试代码块神器
  • VS Code
  • Postman，接口调试必备
  • SourceTree，Git GUI
  • PS

办公软件

  • Microsoft Word，我也不想用啊，别人就要这种文件我能怎么办
  • Excel, 同上
  • PowerPoint，同上
  • TencentMeeting，同上
  • MATLAB，人家就是 nb
  • Parallels Desktop，虚拟机
  • Zoom
  • Spark，邮箱客户端，还行，寻找更好的替代品ing

其他

  • Kana Origin，日语 50 音
  • MWeb Pro，不错的写作工具
  • GoodNotes，主要在 ipad 上用，在电脑上看
  • Wecom 驱动，画画用的
  • 罗技驱动

破解版

大部分软件都是用的正版，能支持正版还是用正版。但有一些由于各种原因不得不用的或者使用频率很低，觉得购买不值的还是找到破解版。

使用破解版存在一定的风险，只是 MacOS 系统下的风险相对较小罢了。
