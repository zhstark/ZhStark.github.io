tips

  • Java 虚拟机将从指定类中的 main 方法开始执行
  • Java 的所有函数都是属于某个类的方法
  • Java 中没有无符号形式的 int、double、short 等
  • char 类型用单引号
  • 声明一个变量之后，必须用赋值语句对变量进行显示初始化，千万不要使用未初始化的变量。
